# stata functions and pipelines
This repository contains self-authored stata functions (mostly for use in genomics analysis). You may get some additional use out of them.

They were written to work on windows 10 and xstata (linux) using STATA 13.1 MP. 

I am relatively new to version control (but embracing it). I will  get better at dev- and stable builds/ bug-fixes as I go along - but I cannot guarantee that when builds are updated that they do not impact other code - e-mail me if they do!. 

The following pages will hopefully be useful in helping you utilise these programs / code-snippets.

* [setting up your ```profile.do``` file](https://github.com/ricanney/stata/blob/master/code/getting_started.md) 
* [available packages](https://github.com/ricanney/stata/blob/master/code/packages.md)
* [cheats, tips and tricks](https://github.com/ricanney/stata/blob/master/code/cheats.md)
