# running the stata genomics packages 
This repository contains self-authored stata programs (mostly for use in genomics analysis). You may get some additional use out of them.
They were written to work on a windows 10 machine and STATA 13.1 MP. I am relatively new to version control (but embracing it). I will  get better at dev- and stable builds/ bug-fixes as I go along - but I cannot guarantee that when builds are updated that they do not impact other code - e-mail me if they do!. 

The following pages will hopefully be useful in helping you utilise theseprograms / code-snippets.


* [getting started](https://github.com/ricanney/stata/blob/master/documents/getting_started.md) - setting up your ```profile.do``` and downloading the packages
* [running the packages](https://github.com/ricanney/stata/blob/master/documents/packages.md) - information on syntax and depndent files
