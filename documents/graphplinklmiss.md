## graphplinklmiss
**description**
* command to plot distribution from *lmiss plink file

**syntax**
 
```graphplinklmiss, lmiss(-filename-) [geno(-geno-)]```
 
* ```-filename-``` the name of the lmiss file \*.lmiss not required
* ```-geno-```     the missingness by marker to be plotted (default = .05)
