## graphplinkhwe
**description**
* command to plot distribution from *hwe plink file

**syntax**
 
```graphplinkhwe, hwe(-filename-) [threshold(-threshold-)]```
 
* ```-filename-``` the name of the hwe file \*.hwe not required
* ```-sd-```       the -log10P to flag in output-file (default = 6)
