## dir2dta
**description** 
* a command to create a file listing all the directories within a directory

**syntax**

```dir2dta, dir(-dir-)```

*  ```-dir-```    root directory to list directories from

**installation**

```net install dir2dta,         from(https://raw.github.com/ricanney/stata/master/code/d/) replace```
