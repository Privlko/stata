## files2dta
**description** 
* a command to create a file listing all the files within a directory

**syntax**

```files2dta, dir(-dir-)```

*  ```-dir-```    root directory to list directories from

**installation**

```net install files2dta,         from(https://raw.github.com/ricanney/stata/master/code/f/) replace```
