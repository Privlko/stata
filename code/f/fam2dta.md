## fam2dta
**description** 
* imports plink \*.fam files into stata 

**syntax**

```fam2dta , fam(-filename-)```

* ```-filename-``` does not require the .fam filetype to be included - this is assumed

**installation**

```net install fam2dta,         from(https://raw.github.com/ricanney/stata/master/code/f/) replace```
