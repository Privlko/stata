## bim2frq_compare  
**description**
* command to compare allele frequencies by reference using plink binaries 

**syntax**

 ```bim2frq_compare, bim(-filename-) ref(-reference-)```

* ```-filenames-``` 	does not require the .bim filetype to be included - this is assumed
*  ```-reference-```	does not require the .bim filetype to be included - this is assumed. 				this is the bim file that others are strand aligned

**installation**

```net install bim2frq_compare, from(https://raw.github.com/ricanney/stata/master/code/b/) replace```
